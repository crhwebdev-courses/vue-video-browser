# video-browser

A project created for Udemy course: Vue JS Essentials with Vuex and Vue Router. This is a simple Youtube video browser. Bootstraped with Vue CLI.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
